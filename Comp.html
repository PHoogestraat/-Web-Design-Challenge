Analaysis

Temperature vs. latitude data was as expected. Temperatures near the equator trend higher than regions with higher latitude values. A second graph (fig.A ) was constructed to overlay data from both hemispheres to contrast trends. Linear regression analysis shows a similar comparison between temperatures for both hemispheres. the slopes for both northern hemisphere and southern hemisphere were similar (North -0.91: South -0.33). R^2 values imply a medium correlation to the data plotted (North 0.77: South 0.18).

Humidity, cloud cover, and wind speed would suggest that a correlation does not exist with latitude bassed on linear regression values . Humidity in the southern hemisphere does show weak correlation and has negative slope which would be consistent with colder climates having less humidity. However the values for the northern hemisphere indicate no correlation and a positve slope. This would not be consistent with the trend. Cloud cover and windspeed also show no correlation with latitude.
