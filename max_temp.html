Analaysis

A plot of latitude vs temperature was obtained from the data set (figure A-1). The data was divided into northern (figure A-2) and southern hemispheres (figure A-3) to obtain linear regression values. Plots for the northern hemisphere had a r^2 value of 0.77 indicating a strong correlation between latitude and temperature. The negative slope also indicated temperatures decrease with increasing latitude which would be consistent with the trend. Temperatures in the southern hemisphere had linear regression value of 0.18 r^2 demonstrating a week correlation. The slope of the line is also negative which would be consistent with the trend.

This data would suggest a correlation exists between an increase of temperature to an decrease of latitude.
